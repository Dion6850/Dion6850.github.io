HDOJ 1005

题目

Problem Description

A number sequence is defined as follows:

f(1) = 1, f(2) = 1, f(n) = (A * f(n - 1) + B * f(n - 2)) mod 7.

Given A, B, and n, you are to calculate the value of f(n).

Input

The input consists of multiple test cases. Each test case contains 3 integers A, B and n on a single line (1 <= A, B <= 1000, 1 <= n <= 100,000,000). Three zeros signal the end of input and this test case is not to be processed.

Output

For each test case, print the value of f(n) on a single line.

题解

第一眼看这个题目，首先想到的是递归算法，但是看到数据n的大小后，有栈溢出的风险，于是就在main函数中用while循环来模拟递归。然而在递交时出现了超时错误，大量的数据计算不能满足题设。于是就只能去找规律来减少计算次数。 因为每次递归涉及的参数是前两个数，因此只要找到不同位置连续两个数对应相同即可。而每个数的情况为0~6，所以两两组合最多49种不同的情况，循环长度最多有98种情况。 但是要注意0的问题。题设中A,B都不会为0，但是因为有mod 7，当A,B是7的倍数时它们的效果与它们是0效果相同。因此，我们需要在程序中加上A,B是否为0的特判。 特判： 当B=0时，从第二个数开始\(f(n)=(f(n-1)*A)\mod 7\)，于是我们只要找到之后某一个\(f(n)\)与等于\(f(2)\)便可以找到循环。 当A=B=0时，除了第一个数和第二个数，其他数都为0。 当A=0时，会使\(f(2n-1)=f(2n),f(2n+1)=(f(2n-1)*B)\mod 7\),可以放在一般情况之中。
